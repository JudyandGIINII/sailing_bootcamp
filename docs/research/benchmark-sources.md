# 세일링 시뮬레이션 벤치마크 조사

- 조사일: 2026-07-18 KST
- 목적: 자연환경 변화에 대응하여 세일 선택·트리밍·조타를 익히는 교육형 세일링 시뮬레이션의 PRD 초안을 위한 기능 및 설계 근거 수집
- 범위: 공개적으로 확인 가능한 게임/시뮬레이터, 공식 세일링 e스포츠 운영 사례, 공공 해양·기상 교육 자료

## 핵심 결론

1. **환경은 장식이 아니라 판단 입력이어야 한다.** 바람, 조류, 조석, 파도·시정은 각각 독립된 표시값이 아니라 보트의 대지속도(ground speed), 표류, 안전 한계, 세일·항로 선택에 설명 가능하게 영향을 줘야 한다.
2. **조작 → 결과 → 원인 → 재시도**의 폐루프가 MVP의 핵심이다. Sailaway처럼 실제 풍황·파도와 세일 공기흐름/트림을 연결하는 접근은 유용하지만, 학습 제품의 초기는 현실 데이터 연동보다 재현 가능한 시나리오와 해설을 우선해야 한다.
3. **교육 모드와 경쟁/자유항해 모드를 분리한다.** World Sailing의 eSailing 사례는 접근성·경쟁 동기를 뒷받침하지만, 본 제품의 1차 성공 기준은 랩타임이나 승패가 아닌 ‘왜 선택이 좋거나 위험했는지’의 이해와 재현 가능한 수행이다.
4. **물리 충실도는 단계적으로 올린다.** 첫 버전에서 난류 CFD·전 지구 실시간 예보·정밀 천문항법을 모두 구현하지 않는다. 검증 가능한 폴라(performance polar), Apparent Wind, 조류 벡터, 조석 기반 수심 여유, 파고/풍속 안전 패널티부터 시작한다.

|---|---|---|---|

## 고려할 환경·선체·교육 모델

### 자연환경 입력

| 범주 | 최소 변수 | 교육적으로 보여줄 인과관계 |
|---|---|---|
| 바람 | 진풍향/풍속, 돌풍, 변화율 | heading + boat speed가 apparent wind를 바꾸며 세일 각도/면적/리프가 달라짐 |
| 조류 | 방향·속도·조석 위상 | 물에 대한 속도와 실제 이동 경로가 달라지며 마크 접근·항로 보정에 영향 |
| 조석 | 수위, 상승/하강, 시간 | 얕은 수역 통과 가능 여부와 안전 여유가 달라짐 |
| 파도 | 파고, 주기, 진행방향 | 선속·조타 안정성·승선 안전 여유에 패널티 또는 한계를 줌 |
| 날씨/시정 | 강수, 운량, 시정, 경보 | 항법 인지, 위험 의사결정, 세일 축소/회항의 근거가 됨 |
| 해역 | 수심, 해안선, 부표/마크, 금지·위험 구역 | 차트 읽기와 충돌·좌초·경계 실패를 설명 가능하게 만듦 |

### 보트·세일 입력

- 선형별 고정된 성능 폴라(풍속/상대풍각 → 목표 선속)로 시작하고, 후속 버전에서 세일별·해상상태별 보정으로 확장한다.
- 최소 조작: helm(heading 또는 rudder), mainsail sheet, jib sheet, sail choice/reef.
- 후속 조작: traveler, vang, cunningham, outhaul, reef 단계, spinnaker/gennaker, weight/balance.
- 각 조작에는 **권장 범위, 현재값, 결과값, 과도조작 경고**를 표시한다. ‘정답 트림’을 암기시키기보다 조건별 트레이드오프를 보여준다.

### 학습 설계

- 레슨은 `상황 브리핑 → 목표/제약 → 자유 조작 → 체크포인트 → 디브리프 → 즉시 재도전`으로 통일한다.
- 디브리프는 결과만 점수화하지 않는다. 관측(풍속/조류) → 판단(세일/코스) → 행동(트림/조타) → 결과(SOG·VMG·안전) 순서의 인과 그래프를 제공한다.
- 난도는 단일 변수(steady wind) → 2변수(wind + current) → 시간변화(gust/tide) → 복합 위험(wave/weather/shallow water)으로 올린다.
- 안전 교육을 우선한다. 위험 상태에서는 벌점보다 회피·감속·리프·귀항 선택과 근거를 긍정적으로 평가한다.

## PRD에 명시해야 할 미결정 사항

1. 우선 플랫폼: 브라우저/데스크톱/모바일/VR 중 MVP 대상.
2. 학습자: 완전 초보, 입문 교육 수강생, 기존 요트 오너/경기자 중 1차 페르소나.
3. 지리·선박: 일반 단일 keelboat, 딩기, 크루저, 한국 연안 특정 해역 여부.
4. 사실성 목표: 물리적 훈련 보조인지, 자격 교육 보조인지, 게임 중심인지. 실제 항해·자격 판단을 대체하지 않는다는 경계.
5. 초기 콘텐츠: 어떤 레슨 3–5개를 MVP 통과 기준으로 삼을지.

## 접근성/출처 품질 메모

- Virtual Regatta의 제품 페이지는 2026-07-18 브라우저 접근 시 Cloudflare 검증으로 상세 기능을 검증하지 못했다. 따라서 위 표의 Virtual Regatta 관련 주장은 World Sailing 공식 파트너십 페이지에서 확인한 범위로만 제한한다.
- RYA와 US Sailing의 특정 교육 과정 URL은 같은 날 404였다. 이 문서에서는 그 URL로부터 교육과정 세부 항목을 인용하지 않았다.
- World Sailing Racing Rules PDF는 후속 ‘레이스 규칙 모드’ 설계 시 별도 조문 검토가 필요하다. 이번 PRD 초안의 MVP 학습 규칙 근거로 사용하지 않았다.
